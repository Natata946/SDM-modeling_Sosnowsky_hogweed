# SDM-modeling_Sosnowsky_hogweed
Моделирование распространения борщевика Сосновского на примере северо-западных административных округов Московской области с помощью метода максимальной энтропии (MaxEnt). Моделирование осуществляется с помощью Python-библиотеки Elapid, разработанной специально для SDM-моделирования на языке Python
