# SDM-modeling_Sosnowsky_hogweed

Моделирование потенциального распространения борщевика Сосновского (*Heracleum sosnowskyi*) на территории северо-западных административных округов Московской области с использованием метода максимальной энтропии — **MaxEnt**.

Проект выполнен на Python с использованием библиотеки `elapid`, предназначенной для species distribution modeling.

## Цель проекта

Цель работы — построить модель пригодности территории для распространения борщевика Сосновского на основе точек присутствия вида и набора пространственных экологических предикторов.

## Что делает проект

В ноутбуке `final.ipynb` реализован полный пайплайн SDM-моделирования:

1. загрузка GeoTIFF-растров из папки `data_predictors`;
2. загрузка CSV-файла с точками присутствия вида;
3. проверка CRS, размеров и сетки растров;
4. генерация фоновых точек;
5. извлечение значений растровых предикторов в точках;
6. обучение модели MaxEnt;
7. оценка качества модели по AUC;
8. расчёт пермутационной важности предикторов;
9. построение карты пригодности территории;
10. сохранение результатов в формате GeoTIFF и CSV.
